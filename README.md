# 🇪🇺 Euro-Inventory Management API
**GDPR-Compliant | Scalable Backend | Clean Architecture**

This is a professional Inventory Management System built with **NestJS** and **TypeORM**, designed to meet European standards for data integrity and security.

## 🚀 Technologies
- **Framework:** NestJS (Node.js)
- **ORM:** TypeORM (PostgreSQL)
- **Database:** Neon.tech (Cloud Managed)
- **Documentation:** Swagger UI (OpenAPI)
- **Environment:** Dotenv for secure credentials

## 🛡️ Business Features
- **Data Portability:** Ready for GDPR data export standards.
- **Audit Ready:** Every entry is timestamped for regulatory compliance.
- **Developer Experience (DX):** Fully documented API with Swagger.

## 📦 Installation & Setup
1. Clone the repository.
2. Run `npm install`.
3. Set your `DATABASE_URL` in the `.env` file.
4. Run `npm run start:dev`.
5. Access the interactive documentation at `http://localhost:3000/api/docs`.
